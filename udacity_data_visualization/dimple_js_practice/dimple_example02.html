Summary: Purpose of this project is to visual Titanic survival data using dimple.js framework.

Survival rate by Age

Observation: Age doesn't seem to have much impact on the survival except for children under the age of 5. The survival rate actually decreases for age group 5-10 before it increases again slightly for age-group 10-15.

Survival rate by Sex and Class

Observation: Females from 1st and 2nd class have > 90% survival rate whereas females from 3rd class has survival rate of around 50%. Among males, 1st class passengers has higher rate then 2nd and 3rd class. But the 1st class male paseengers survival rate is nowhere near the 1st class females.

Conclusion

Based on this dataset, below subgroups had better chance of survival..

  • females passengers
  • class 1 passengers
  • children less then 5 years
Below subgroups had lesser chance of survival...
  • male passengers
  • class 3 passengers
  • older passengers

However, these results are based on the data present and there would be exceptions to these findings.This dataset just had 891 records out of the total of 2,224 passengers who were there on RMS Titanic. There might be other factors impacting the survival outcome. So we conclude that this is a very limited analysis and might have different results if we had a different sample set or the whole data.
